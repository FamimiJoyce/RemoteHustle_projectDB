Project Description
This repository contains the Remote Hustle database project, including:

- SQL script (RemoteHustle_DB.sql) to create tables, insert data, and create views.
- ER diagram (RemoteHustle_ERDiagram.png) showing table relationships.
- Project description (RemoteHustle_Description.docx) explaining features, tech stack, and usage.

 How to Use
1. Open RemoteHustle_DB.sql in your SQL client (Supabase SQL editor, pgAdmin, etc.).
2. Run the script to create the database and views.
3. Refer to the ER diagram for table relationships.
4. Read the project description for context.
